## 功能

- 浏览首页推荐、连载更新、排行榜
- 搜索漫画，按标签分类筛选
- 收藏夹
- 漫画详情与章节列表
- 章节阅读器（缩放、翻页、阅读设置）
- 离线下载
- 阅读历史
- 章节评论
- 账号登录
- 深色 / 浅色主题切换（圆形扩散动画）
